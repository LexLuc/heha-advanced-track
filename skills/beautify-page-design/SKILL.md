# 美化已有页面

在保留原有功能和结构的基础上，大幅提升页面的视觉效果和交互体验，避免千篇一律的"AI味"设计。

## 步骤

### 1. 读取现有页面

- 确认用户要美化的页面文件路径（优先使用用户当前打开的文件）
- 读取页面完整代码，理解现有的功能、结构和样式

### 2. 诊断现有设计

分析页面当前的设计问题，从以下维度评估：
- **字体**：是否使用了 Arial、系统默认字体等无特色字体？
- **配色**：是否单调、缺乏层次？是否用了烂大街的紫色渐变白底？
- **布局**：是否过于平铺直叙、缺乏节奏感？
- **动效**：是否缺少动画和交互反馈？
- **背景与细节**：是否只有纯色背景、缺乏氛围感？
- **整体风格**：是否缺少明确的设计方向？

将诊断结果简要告知用户。

### 3. 确定美化方向

根据页面现有的内容和气质，选择一个**大胆且明确**的美学方向：
- 极简克制、极繁华丽、复古未来、赛博朋克、自然有机、奢华精致、童趣玩味、杂志编排、粗野主义、装饰艺术、柔和粉彩、工业实用……
- 关键是**有态度**，而不是面面俱到的平庸

将美化方向告知用户，获得确认后再动手。

### 4. 实施美化

在**不改变原有功能逻辑**的前提下，从以下方面提升设计：

- **字体**：选择有辨识度的字体搭配（标题字体 + 正文字体），避免 Inter、Roboto 等泛用字体。用 Google Fonts 引入
- **配色与主题**：建立统一的 CSS 变量色彩体系，主色突出、点缀色鲜明，避免胆怯的均匀分布
- **动效与交互**：添加页面加载动画、hover 状态、滚动触发效果等。HTML 页面优先用纯 CSS 方案，React 项目可用 Motion 库。重点打磨入场动画（配合 animation-delay 做错落感）
- **空间构成**：打破死板布局，引入不对称、重叠、对角线流动、破格元素、大面积留白或有控制的密集感
- **背景与视觉细节**：用渐变网格、噪点纹理、几何图案、透明叠层、戏剧性阴影、装饰边框、颗粒叠加等手法营造氛围和层次

**禁止**产出千篇一律的 AI 风格：不要用烂大街的字体、不要用紫色渐变白底的老套配色、不要用毫无特色的模板式布局。

**重要**：实现复杂度要匹配美学方向。极繁风格需要大量动效和细节堆叠；极简风格则需要克制精准，在字距、留白、微妙细节上下功夫。

### 5. 展示改动

美化完成后，简要说明做了哪些改动，让用户对比效果。
